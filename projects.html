SENIOR PROJECTS:

    2012-2013:

  • Find Location With Video: Finding Location With Video is a Mobile Application for iOS platform. The application is for finding location from anywhere inside Engineering Building.

  • Where's The Instructor/Advisor: The purpose of the project is to design and develop an Android app which enables Instructors to broadcast their whereabouts (at office, class, lab, out of campus, etc.) to their students.

  • Simulating DDoS Attacks: The purpose of the project is to design and develop a simulation in which DDoS attacks and their effect on a system can be observed.

  • Traffic Reports for the Roads to Atilim University Campus : The purpose of the project is to design and develop an Android app which creates a Driving Social Network to collect traffic events (accidents, ice, construction, etc.) from this community and to share collected information to the users instantly.

  • Book&Review: The purpose is to help the users to search the books information and comments in an easy way with help of smart phones.

  • Interactive COMPE101 : The aim of our project is, creating a web based multimedia application about introducing the computer parts, teaching principles of operation and function of this parts and providing information about the implementation of computer hardware changes to the COMPE 101 students.

  • Alerts for Atilim University: The purpose of the project is to design and develop an Android application for disseminating/receiving alerts for news, events, announcements, libraries, emergency situations and weather conditions submitted by the university administration to the application users.
