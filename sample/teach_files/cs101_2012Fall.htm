COMPE 101

Course Syllabus

 

COMPE 101 Introduction to Computer Engineering and Programming                                         (2-2)3

 

Catalog Data

Basics of Information Systems, computer software, computer hardware, secondary storage, Input/Output, internet, TCP/IP, ftp, telnet, networks, and basic programming concepts. Hands-on experience of application software and internet through lab sessions.

 Prerequisite by Topic

·          Numbering systems

·          Problem solving skills

 

Method for Assessing Student Knowledge of Prerequisite Topics

During the first lecture an oral questionnaire will be given to the students.

 

Goals

Provide students with the basis of information systems; gain experience in basic applications software, Internet and basic programming.

 

Objectives

1.        To learn computer types and information systems

2.        To learn computer software and hardware.

3.        To learn how to use word processor, spreadsheet and  presentation  softwares

4.        To learn basics of internet, browsers , TCP/IP, telnet, ftp

5.        To learn basic programming concepts

 

Topics

1.        Introduction to Computer Systems (2 weeks)      

2.        Computer Software (2 weeks)                  

3.        Computer Hardware (1 week)                                  

4.        Secondary Storage (1 week)                                    

5.        Input/Output (1 week)                                                                                             

6.        Networks      (1 week)                                

7.        Internet (1 week)                                                                       

8.        Introduction to Programming   (5 weeks)              

 

Computer Usage

Weekly laboratory sessions (2 hours/week) will be held under the supervision of the instructor and the lab assistant.

 

Laboratory Projects

Students will use Office applications and Internet and get hands-on experience through laboratory assignments.

 

Contribution of Professional Component

1. Math or Basic Science                                   0 credits

2. Engineering Topics and Design                  3 credits

3. General Education                                          0 credits
